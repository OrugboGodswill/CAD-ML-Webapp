CADPredict Documentation

Comprehensive guide to understanding and using our AI-powered coronary artery disease prediction system.

Important Disclaimer

CADPredict is an educational tool, not a medical device. It does not provide medical advice, diagnosis, or treatment. Always consult a qualified healthcare provider for any concerns about coronary artery disease or related issues.

Overview

CADPredict is an AI-powered web application designed to assess the risk of coronary artery disease (CAD) based on user-entered health metrics. Using a trained machine learning model (SVM with linear kernel), the app provides risk predictions and "explainability" via feature coefficients, helping users understand which health factors contribute most to their estimated risk.

The goal is not to replace medical judgment, but to raise awareness, promote preventive behaviors, and enable users to make more informed health decisions.

What CADPredict Does

Data Collection

Accepts input of health factors via a user-friendly form including age, cholesterol, blood pressure, smoking, adiposity, family history, stress behavior, BMI, and alcohol use.

Data Processing

Preprocesses the inputs (scaling, encoding) exactly as used in model training to ensure accurate predictions.

ML Prediction

Uses a machine learning model (Support Vector Machine) to predict whether a user is at high risk or low risk for CAD.

Feature Analysis

Highlights which features in the user's input contributed most strongly to the prediction, based on model coefficients.

Key Features

Risk Prediction

Input your health metrics and get a prediction: High Risk or Low Risk.

Probability Score

See how likely the model believes CAD is present (a percentage), not just a binary yes/no.

Feature Importances

The model shows which features (age, cholesterol, etc.) had the highest positive or negative influence on the prediction.

Responsive UI

Designed to work well on desktop & mobile. Clear, intuitive fields; sliders or radio buttons where appropriate.

How It Works (Technical Details)

Data Preparation

The model was trained on a dataset with features including sex, systolic blood pressure, tobacco usage, cholesterol, adiposity, family history, stress prone behavior, BMI, alcohol consumption per year, age, and a binary target whether CAD is present.

Preprocessing

Data was scaled (with StandardScaler) so each feature has mean zero and standard deviation one. Categorical data (like family history) was encoded to binary.

Model Architecture

A linear Support Vector Machine (SVM) with a linear kernel, possibly balanced class weights to handle any imbalance, trained and validated on hold-out data.

Limitations & Caveats

The model is only as good as the data used to train it; biases in the training data can create unexpected behavior.

Rare medical conditions or risk factors outside the model's features aren't captured.

Prediction tools like CADPredict do not replace professional diagnosis, lab tests, or imaging studies. Always consult a doctor.
